# InsightAI - Universal Analytics Platform

AI-powered data analysis supporting CSV, Excel, PDF, and Word files.

## Features
- 📊 Multi-format file support (CSV, Excel, PDF, Word)
- 🤖 AI-powered analysis using Groq
- 📈 Interactive charts and visualizations
- 🎤 Voice interaction
- 🔐 Google Authentication
- 💾 MongoDB data storage

## Usage
1. Upload your data file (CSV, Excel, PDF, or Word)
2. Click 'Analyze Data' to get AI-powered insights
3. Interact with visualizations and charts
4. Use voice commands for hands-free analysis
